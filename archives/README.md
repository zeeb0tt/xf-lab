# XenForo archives

Put your licensed XenForo ZIP downloads in this directory.

Name them exactly like:

```text
xenforo-2.1.15.zip
xenforo-2.2.19.zip
xenforo-2.3.10.zip
```

Then start an instance with:

```bash
./scripts/xf-up 2.3.10 8.3
```

The script will look for:

```text
archives/xenforo-2.3.10.zip
```

Do not commit XenForo ZIPs to a public repository. They are licensed software.
